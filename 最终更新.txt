   • 界面更简洁

🎯 核心功能
===========

交易时间检测：
  • 工作日 09:30-11:30, 13:00-15:00
  • 自动识别周末
  • 显示下次开盘时间

智能刷新策略：
  • 开盘时：每1分钟刷新价格
  • 休市时：只显示收盘价，不刷新
  • 自动切换，节省API调用


💡 使用方法
===========

调整大小：
  方法1：鼠标滚轮（最方便）
  方法2：右键 → 调整大小 → 选择预设
  方法3：拖动右下角

查看状态：
  • 鼠标悬停显示详细信息
  • 开盘：显示"市场开盘中"
  • 休市：显示"市场休市"和下次开盘时间

修改代码：
  • 双击气泡
  • 或右键 → 设置代码


📊 显示信息
===========

悬停提示（开盘时）：
  • 代码
  • 当前价格
  • 昨收价
  • 涨跌金额和百分比
  • "每1分钟自动刷新"

悬停提示（休市时）：
  • 市场休市
  • 当前时间
  • 下次开盘时间
  • "显示最新收盘价"
  • "不会自动刷新"


🎨 视觉效果
===========

颜色方案：
  涨：半透明绿色边框 + 深绿文字
  跌：半透明红色边框 + 深红文字
  平：半透明白色边框 + 黑色文字

背景：
  半透明白色毛玻璃效果
  融入桌面背景

大小范围：
  最小：50x50
  默认：80x80
  最大：300x300


⏰ 交易时间示例
===============

周一至周五：
  09:30-11:30 开盘 → 自动刷新
  11:30-13:00 午休 → 停止刷新
  13:00-15:00 开盘 → 自动刷新
  15:00后     休市 → 停止刷新

周六日：
  全天休市 → 不刷新
  显示"下次开盘：周一 09:30"


💾 保存的设置
=============

%AppData%\StockBubble\ 文件夹：
  • config.txt - 代码
  • size.txt - 窗口大小
  • error.log - 错误日志


🚀 使用建议
===========

1. 首次使用
   • 输入测试代码：sz000001 或 sh600036
   • 调整到合适大小（滚轮）
   • 拖动到喜欢的位置

2. 日常使用
   • 无需任何操作
   • 开盘自动刷新
   • 休市自动停止
   • 鼠标悬停查看详情

3. 出现错误
   • 鼠标悬停查看原因
   • 右键 → 查看错误日志
   • 双击修改代码
